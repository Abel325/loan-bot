# Waafi Loan Assistant Bot 🤖

Telegram loan application bot built with Python and SQLite.

## Features

✅ Welcome message  
✅ User registration  
✅ Save user details  
✅ Loan application  
✅ Loan amount selection  
✅ Loan status checking  
✅ Loan requirements information  
✅ Support contact  
✅ SQLite database storage
